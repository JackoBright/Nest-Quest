Welcome to the NestQuest Website!

Overview

  • Users holds information about every user in the database.
  • Seekers holds information about users searching for a place to live.
  • Tenants holds information about users who already live in a property.
  • Information Requests holds information about infor request sent out by seekers who have requested information about a property.
  • Reviews holds information about property reviews written by users.

Specifications

  • Insert / Select(Read) implemented for all entities.
  • Delete / Update implemented for Users and Properties.
  • Search implemented for Users.
